            MySQL backups

            Automatic backups run daily while DevTent is open, and before MySQL stops. Backups are kept for 7 days in data/backups/mysql/.

              Export / import

              Export your projects, profiles, configs, and database data to move DevTent to another folder or machine. Runtimes in bin/ are excluded by default (reinstall via Quick Add).

              Quit behavior

              Disable for faster quit. Updates always use fast exit so the installer can replace files.

              Import environment

              The only place to import from Laragon or another local environment. Copies projects from www/, php.ini files, databases, and runtimes. Never runs during install or first-run setup — use this panel when you choose to import.
